# 🦅 Striving Eagle

> **Fly Higher in Style — Trendy Fashion & Accessories in Malawi**

Striving Eagle is a modern e-commerce platform for fashion-forward individuals in Malawi, offering contemporary streetwear and accessories with a focus on quality, style, and accessibility.

## 🌟 Features

### 👕 **Fashion Collections**
- **Men's Collection** - Premium denim, modern streetwear, casual wear
- **Women's Collection** - Elegant fashion, contemporary styles, accessories
- **Unisex Options** - Versatile pieces for all genders
- **Latest Arrivals** - Regularly updated with new trends

### 🛒 **Shopping Experience**
- **Product Categories** - Easy navigation by gender and style
- **Product Details** - High-quality images, descriptions, pricing
- **Size Selection** - Multiple size options with availability
- **Wishlist & Cart** - Save favorites and manage shopping cart

### 💳 **Payment & Delivery**
- **Multiple Payment Methods**:
  - Airtel Money
  - TNM Mpamba
  - Bank Transfers (Standard Bank, NBS, National Bank)
  - Cash on Delivery (Lilongwe/Blantyre)
- **Nationwide Delivery** - Across Malawi
- **7-Day Return Policy** - Easy exchanges if size doesn't fit

### 📱 **Customer Service**
- **WhatsApp Support** - 24/7 customer assistance
- **Phone Support** - Direct contact during business hours
- **Size Guide** - Help finding the perfect fit
- **Style Recommendations** - Personal fashion advice

### 🔐 **Admin Dashboard**
- **Product Management** - Add, edit, delete products
- **Order Management** - Track and fulfill customer orders
- **Customer Support** - Handle customer inquiries
- **Analytics** - Sales and performance insights

## 🚀 Quick Start

### **Prerequisites**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- WhatsApp for customer support

### **Installation**

#### **Option 1: Live Preview**
1. Visit the deployed website: [https://striving-eagle.pages.dev](https://striving-eagle.pages.dev)
2. Start browsing and shopping immediately

#### **Option 2: Local Development**
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/striving-eagle.git
   cd striving-eagle
   ```

2. Open `index.html` in your web browser:
   ```bash
   # Simply double-click index.html or use:
   start index.html
   ```

3. For admin access:
   - Navigate to `login.html`
   - Use admin credentials (contact owner for access)
